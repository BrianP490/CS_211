This program checks to see if the users input has balanced parameters i.e.  "(,),[,],{, and }" using a stack

ex. 
(9*0)       BALANCED
[5+(5+1)]   BALANCED

[5          UNBALANCED
6-4]        UNBALANCED

To run the program, navigate to the downloaded Part1Main directory and run the command:
./pt1.exe

in a terminal
